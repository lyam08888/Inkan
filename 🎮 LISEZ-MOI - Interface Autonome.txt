🎮 DOFUS SHIVAS - INTERFACE AUTONOME
===================================

🚀 NOUVEAU : TOUT FONCTIONNE DEPUIS L'INDEX.HTML !
-------------------------------------------------

Cette interface est maintenant 100% autonome :
• Aucun serveur web externe requis
• Tous les scripts sont générés automatiquement
• Fonctionne directement dans votre navigateur
• Télécharge tous les fichiers nécessaires

📁 DÉMARRAGE ULTRA-SIMPLE :
--------------------------
1. Double-cliquez sur "🎮 DOFUS HORS LIGNE - TOUT EN UN.bat"
2. L'interface s'ouvre dans votre navigateur
3. Suivez les étapes dans l'interface
4. Tous les scripts sont téléchargés automatiquement !

🎯 FONCTIONNEMENT DE L'INTERFACE :
---------------------------------

📋 ÉTAPE 1 : Configuration automatique
• Cliquez sur "Configuration automatique"
• L'interface télécharge tous les scripts :
  - install-java.ps1 (installation Java si nécessaire)
  - setup-database.ps1 (configuration base de données)
  - compile-project.ps1 (compilation du projet)
  - setup-config.ps1 (configuration des fichiers)

🚀 ÉTAPE 2 : Exécution des scripts
• Exécutez les scripts téléchargés dans l'ordre
• Chaque script est autonome et guidé
• Suivez les instructions à l'écran

🎮 ÉTAPE 3 : Démarrage du serveur
• Cliquez sur "Démarrer le serveur" dans l'interface
• Téléchargez et exécutez start-shivas-server.ps1
• Le serveur Dofus démarre automatiquement

📥 ÉTAPE 4 : Configuration client
• Téléchargez config.xml depuis l'interface
• Placez-le dans votre dossier client Dofus
• Lancez Dofus et sélectionnez "localhost"
